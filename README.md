## How to Play
Drag the kitten to a safe area (under the table or bed).
Follow the on-screen instructions: Drop, Cover, Hold On.
Survive the earthquake to win.

## Learning Intent
This game teaches children what to do during an earthquake using the
"Drop, Cover, Hold On"

## Engine
Custom JavaScript 2D game engine (canvas-based).

## Controls
Mouse / touch drag.

## Assets
Simple shapes draw by code.
